# Проектная работа Mesto
Проект "Mesto" - это веб-ресурс, который представляет собой интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки.

## Описание проекта
Сайт "Mesto" альтернатива Instagram. Здесь вы можете как создать свой профиль, где будете делиться своими фотографиямис мест своего пребывания, так и наблюдать за профилями других людей.

## Как использовать
Для начала следует загеестрироваться на сайте, а после наслаждаться

## Технологии
Сайт "Mesto" создан с использованием следующих технологий:

JavaScript
HTML5
CSS

## Установка

1. Клонируйте репозиторий: https://github.com/yourusername/mesto-project-ff.git
2. Открыть файл index.html в вашем браузере.
